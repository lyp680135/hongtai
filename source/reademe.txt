下载该项目，在搭建好nodejs开发环境的基础上
执行npm install 命令，加载模块。

注意的是：其中xy-mint-ui模块包存放在我们自己建立的私有仓库(采用sinopia 搭建)，仓库地址为：http://121.43.224.48:8000/

所以先执行：
npm install xy-mint-ui --registry http://121.43.224.48:8000
然后再执行其他模块下载（使用淘宝镜像快一点）
npm install --registry=https://registry.npm.taobao.org
